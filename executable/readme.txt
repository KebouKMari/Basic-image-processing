1. Pour compiler le programe dans le terminal, placer vous dans le dossier en question et tapez la commande : gcc *.c -o execute

2. Pour executer , taper dans votre terminal, la commande pour avoir les differentes fonctions que vous pouvez utiliser : ./execute info

3.Par exemple pour egaliser une image vous tapez ceci : ./execute egal nomdufichierAtransformer.pgm  nomfichierdefin.pgm

NB : Nous travaillons qu'avec des images en niveau de gris (.pgm) 
     Lorsque nous manipulons les images binaires, nous utilisons les fichiers(.pbm)
